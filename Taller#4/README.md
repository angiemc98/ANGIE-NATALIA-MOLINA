# Hospital-management-nestjs_Taller-4

## Stack Tecnologico
- NestJS
- Compodoc
- Swagger
- Jest 
- Git

## Instalación
- Clonar el repositorio
https://github.com/angiemc98/Hospital-Management.git 
- cd Hospital-Management

## Instalación de dependencias
- `npm install`

## Ejecutar el servidor
Modo de Desarrollo
- `npm run start:dev`

## Modo de Uso
El sistema permite la gestión de:

- Pacientes
- Médicos
- Citas médicas
- Historial clínico, etc

Cada módulo cuenta con sus rutas, controladores y servicios documentados en Compodoc.
La API expone endpoints RESTful con tipos definidos en los DTOs correspondientes

## Modo de Testeo
- `npm run test`

## Documentación
- `npm run start:swagger`

## Compodoc
- `npm run start:compodoc`

## Coverage
- `npm run start:coverage`
